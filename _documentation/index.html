HTML Structure

This is a responsible theme layout with various number of columns (anything between 1-12), which allows to create custom combination of columns by using specific class selectors. The theme is based on popular Twitter Bootstrap interface, which comes with support for responsive design, HTML5, CSS3 and many other user interface components. Since twitter bootstrap is very well integrated into this theme, you can use any of it's components with ease. More information about using twitter bootstrap can be found on http://twitter.github.com/bootstrap/. 

All of the information within the main content area is nested within a div with a class of  "row". 

Within "row," 12 columns can be easily divided into any number of sections. 

To create 2 sections with the same width, the code would be written as follows:

  1. <div class="row">
  2. <div class="col-md-6">...</div>
  3. <div class="col-md-6">...</div>
  4. </div>

Since 6+6 = 12, these columns sum up to the full row. Alternatively, when creating 3 sections with the same width, the code needs to be changed like this:

  1. <div class="row">
  2. <div class="col-md-4">...</div>
  3. <div class="col-md-4">...</div>
  4. <div class="col-md-4">...</div>
  5. </div>

Using four div elements with the class of "span3" will respectively result in four columns with the same width. 
Creating columns with different width is as easy as splitting the number 12 into desired sections. 

1/3 layout can be achieved by following code

  1. <div class="row">
  2. <div class="col-md-4">...</div>
  3. <div class="col-md-8">...</div>
  4. </div>

CSS Files and Structure

This theme comes with 5 CSS files:

1. bootstrap.min.css
2. bootstrap-responsive.min.css
These are base twitter bootstrap CSS files, which defines various elements like  columns, navigation, tabs, icons, forms and buttons. These files also take care of responsiveness of this theme, so it can be easily viewed on mobile devices or tablets. 

3. bootstrap-override.min.css
This file is not an official part of twitter bootstrap theme, it contains few overrides of media queries of original theme, which were required for this theme to work correctly. The most important chage it, that while keeping twitter bootstrap's responsive functionality, it restricts content section of our website to be at most 1024px width.

3. font-awesome.css (comes with font-files)
This file includes a custom font which is being used for all icons which you can see in this theme. More information about Font Awesome 100% scalable icon pack can be found at http://fortawesome.github.com/Font-Awesome/. 

4. style.css
This is the most important style for our theme, and to make any changes to this template, you should only need to modify this file. The file is divided info dirrefent sections, while every section take name from a subpage respectively. 

 

/* Base */ - some basic styling of our theme. 
/* Classes */ - few classes which are being used accross all theme's files. 
/* Footer */
/* Tabs */
/* Navigation */
/* Carousel */
/* Hero unit (page headings) */
/* Pricing */
/* Portfolio */
/* Our team */
/* Explore features */
/* Special elements */ (e.g. - ribbon as can be seen on pricing tables)
/* Thumbnails */

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

JavaScript

This theme imports three Javascript files.

  1. jquery-1.11.3.min
    jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. bootstrap.min
    Twitter bootstrap base JavaScript file, which allows us to use many javascript components like tabs, accordion menus, dropdowns, tooltips and popovers, alert messages and so on. ore can be found at http://twitter.github.com/bootstrap/javascript.html
  3. jquery.isotope.min.js
    Which is being used to filter our portfolio items
  4. jquery.hotkeys
    For keyboard shortcuts functionality (e.g. navigating to previous/next project by pressing Left/Right arrow)

  5. functions.js
    The most important file for our theme, which defines some basic javascript functions. 

    This file is divided into few different sections, while every section cares about different functionality. 

    Twitter bootstrap - carousel, tooltip, popover 
    Initialization of some twitter bootstrap javascript plugins

    Thumbnails
    Zoom and link functionality of out thumbnails (+ and link icons which appear after you move your mouse over thumbnail - e.g. portfolio item)

    Responsible navigation
    Creates a collapsible top navigation for mobile devices

    Portfolio 
    Basic functionality for filtering portfolio items. 

    Keyboard shortcuts
    Initialization of keyboard shortcuts. You can easily define your own shortcuts by using this code:

    <a rel="shortcut" data-key="k" href="http://google.com">Press "k" to visit google.com</a>

    This code will result in navigation your visitor to google.com after pressing "k" key.
